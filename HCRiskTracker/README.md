# HC Risk Tracker

HC Risk Tracker is a lightweight addon designed for World of Warcraft Hardcore.

It provides a real-time risk indicator based on:
- Player health and resources
- Missing essential buffs
- Missing pet (Hunter / Warlock)
- Missing poisons (Rogue, after learning "Poisons")
- Nearby hostile threats (before combat)
- Elites and indoor danger

## Features
- Risk bar with threat counter
- Tooltip with detailed scan information
- No chat spam (alerts only at high risk)
- Drag & lock position
- Automatic FR / EN localization
- Hardcore-friendly (no sounds, no blinking)

## Commands
/hcrt lock  
/hcrt unlock  
/hcrt reset  

## Designed for Hardcore
The addon focuses on decision-making, not panic.
